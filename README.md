#  Secure Login System with User Role Management

This project was developed as part of the Cyber Security Internship assigned by **Vault-Tec Security**.

The objective of this project was to design and implement a secure web-based authentication system incorporating user authentication, authorization, and role-based access control (RBAC), while applying core cybersecurity best practices in a practical development environment.

---

##  Project Objective

To build a secure login system that includes:

- User Registration & Login
- Secure Password Hashing
- Session Management
- Role-Based Access Control (Admin/User)
- Protection against common web vulnerabilities
- Proper testing and documentation

This project demonstrates secure software development principles combined with version control and structured implementation.

---

## 🛠 Technology Stack

- Python 3
- Flask
- Flask-Login
- Flask-WTF
- Flask-SQLAlchemy
- SQLite
- HTML / CSS / JavaScript
- bcrypt (for password hashing)
