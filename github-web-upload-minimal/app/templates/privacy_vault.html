Security

Privacy Vault

This section is dedicated to keeping user documents, photos, and personal records private by default and securely protected.

Storage Security

  • Files are encrypted using AES-256 before storage.
  • Only allowed file formats are accepted.
  • Maximum upload size is restricted (5MB).
  • Optional malware scan can be enabled for uploads.

Access Control

  • Achievements and goal posts support private/public visibility.
  • Private content is only visible to its owner (or admins).
  • Session checks and role checks protect sensitive operations.
  • CSRF protection is enabled on forms.

Account Security

  • Passwords are hashed using bcrypt.
  • Rate limiting helps block brute-force attempts.
  • Email verification and 2FA options are supported.
  • Secure cookies and strict security headers are applied.

Privacy Recommendation

  • Use private visibility for personal certificates or IDs.
  • Avoid uploading unnecessary sensitive data.
  • Enable 2FA from Security Settings for extra protection.
  • Use strong passwords and rotate them regularly.
